# 📱 에벤에셀 전기 거래 장부 - PWA 설치 가이드

## 🎯 PWA(Progressive Web App)란?

웹사이트를 스마트폰이나 PC에 앱처럼 설치하여 사용할 수 있는 기술입니다.

## 📲 모바일에서 홈 화면에 추가하기

### Android (Chrome)
1. Chrome 브라우저로 웹사이트 접속
2. 우측 상단 **⋮** (메뉴) 클릭
3. **"홈 화면에 추가"** 선택
4. 앱 이름 확인 후 **"추가"** 클릭
5. 홈 화면에 앱 아이콘이 생성됩니다

### iPhone (Safari)
1. Safari로 웹사이트 접속
2. 하단 **공유** 버튼 (□↑) 클릭
3. **"홈 화면에 추가"** 선택
4. 앱 이름 확인 후 **"추가"** 클릭
5. 홈 화면에 앱 아이콘이 생성됩니다

## 💻 PC에서 설치하기

### Chrome / Edge
1. 브라우저로 웹사이트 접속
2. 주소창 우측의 **설치(⊕)** 버튼 클릭
3. **"설치"** 클릭
4. 데스크톱에 앱으로 설치됩니다

## ✨ PWA 앱 장점

- ⚡ **빠른 실행**: 앱처럼 바로 열림
- 📱 **전체 화면**: 브라우저 UI 없이 깔끔한 화면
- 🔄 **오프라인 지원**: 인터넷 없어도 기본 기능 사용 가능
- 🎨 **네이티브 앱 느낌**: 앱 아이콘, 스플래시 화면
- 🔔 **독립 실행**: 일반 앱처럼 작업 전환 가능

## 🛠️ 기술 구성

- **manifest.json**: 앱 메타데이터 및 설정
- **Service Worker**: 오프라인 지원 및 캐싱
- **아이콘**: 192x192, 512x512 크기
- **테마 색상**: #667eea (보라색)

## 📝 참고사항

### 아이콘 이미지 생성
SVG 파일을 PNG로 변환하려면:

1. **온라인 변환기 사용**
   - https://svgtopng.com/
   - icon-192.svg → icon-192.png (192x192)
   - icon-512.svg → icon-512.png (512x512)

2. **또는 디자인 도구 사용**
   - Figma, Adobe Illustrator 등에서 열어서 PNG로 내보내기

### Service Worker 주의사항
- HTTPS 필수 (localhost는 예외)
- 캐시 업데이트 시 버전 번호 변경 필요 (sw.js의 CACHE_NAME)
